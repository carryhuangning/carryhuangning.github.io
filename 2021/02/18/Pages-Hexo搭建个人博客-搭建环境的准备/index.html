基于github Pages+Hexo搭建个人博客--搭建环境的准备

发表于 2021-02-18
本文字数： 1.2k 阅读时长 ≈ 1 分钟

上期我们详细介绍了什么是个人博客，接下来，我将会专门讲述如何利用githubpages+hexo简单地搭建我们的个人博客，希望这一系列文章可以给希望拥有个人博客的你提供建议。

在正式开始之前，我们首先需要搭建环境的准备工作：

git环境

node.js环境

Github账户的注册及相关设置

一、安装git

接下来，小编会以git version 2.30.1.windows.1为例，向大家介绍如何安装git

git官网下载地址：https://git-scm.com/

需要注意的是，安装路径最好不要默认在C盘，除非你的C盘足够strong；在安装过程中，直接点击下一步即可；最后，安装成功后，右键任意空白处点击git bash here，在命令行中输入git –version,回车查看你的版本号以判断你是否安装成功。

二、安装node.js

以node.js14.15.5版本为例，进入node.js官网下载：https://nodejs.org/zh-cn/[](https://nodejs.org/zh-cn/)

同样注意不要装在C盘，安装过程一路点击”NEXT”即可，最后，在git bash命令行中输入node -v即可查看node.js版本号。

三、注册Github账号并creat “new repository”

进入官网https://github.com/注册一个账号
注册账号需要你的Username（用户名）、Email address(邮箱）、Password(密码)
注册好之后你会进入你的主页面，点击右上角头像旁New repository即可创建你的新仓库来存放你的代码，为了获得Github的pages，仓库名称一定要命名为**(your username).github.io** 例如我的username是carryhuangning,命名carryhuangning.github.io，至于Add a README file最好勾选以便添加介绍性文本来介绍你的仓库。

随后进入你的仓库页面

点击Settings,x下拉找到Github pages,这是Github自动为你生成的网页，点击Launch automatic page generator

choose一个你喜欢的主题，这样子就生成一个简单的网页了

以上步骤做完，在网页地址栏输入（your username).github.io即可访问你刚刚创建的网页了，不过，我们的博客并不是发布在这里的，使用Github是为了方便本地、云端数据同步以及域名解析等后期工作的方便，此外，我们还相当于白嫖一台国外的服务器，对于经济尚未独立的我们，这何尝不是一种好办法。

夜已深，今天的文章就到这里，下一篇我将会从hexo 开始，继续向大家展示如何利用GitHub pages+hexo搭建博客。多谢大家的支持，晚安，祝好梦！
